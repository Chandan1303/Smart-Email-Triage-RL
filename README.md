# Smart Email Triage RL

A reinforcement learning environment where an AI agent classifies emails, sets priorities, and earns rewards for correct decisions.

## Features
- 3-step multi-step decision process (Classify → Priority → Act)
- Score dashboard with accuracy tracking
- Email summarization and auto-reply suggestions
- Difficulty levels: Easy / Medium / Hard
- Q-Learning agent training
